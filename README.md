# IoT based Smart Mailbox System
서버와 세 개의 디바이스를 통해 사용자가 언제, 어디서나 우편함의 우편물의 유무를 알 수 있는 IoT시스템을 구현

# Usage Tech
HTML, JSP, JavaFX, C, Apache TomCat, NodeMCU, RaspberryPi 3+, Android Studio
